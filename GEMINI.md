# プロジェクト概要: Kengo Hoi — Portfolio

このプロジェクトは、**Astro v5** を使用して構築された、個人のポートフォリオサイトです。
シングルページアプリケーション（SPA）のようなスムーズな閲覧体験を提供し、スキルセット、経歴、制作実績（Works）を1ページに集約して表示しています。

## 1. プロジェクトの概要と技術スタック

*   **フレームワーク:** [Astro](https://astro.build/) (v5.9.2)
    *   静的サイト生成（SSG）による高速なパフォーマンスを実現。
*   **言語:** TypeScript
*   **スタイリング:**
    *   **CSS Variables:** テーマ（ライト/ダークモード）の切り替えや共通カラーの管理。
    *   **Scoped CSS:** コンポーネント単位のスタイル管理による競合防止。
*   **コンテンツ管理:**
    *   **Content Collections:** `src/content/work` 内の Markdown ファイルを使用して制作実績を管理。Zod によるスキーマ定義（`src/content.config.ts`）でデータ構造を保証しています。
    *   **Skills:** `src/components/SkillsList.astro` 内の配列データとして定義し、カテゴリ別に自動分類して表示。

## 2. ディレクトリ構成と役割

主要なディレクトリとファイルの役割は以下の通りです。

*   **`src/pages/`**:
    *   `index.astro`: **サイトのトップページ**。各セクションコンポーネントを集約し、動的な背景演出を実装。
    *   `work/[...slug].astro`: Markdown から個別ページを生成するルート（現在はトップページのモーダル表示がメイン動線）。
*   **`src/layouts/`**:
    *   `BaseLayout.astro`: `<head>`, `Nav`, `Footer` など、全ページ共通の枠組みを提供。
*   **`src/components/`**:
    *   `WorksSection.astro`: 制作実績セクションの本体。Content Collections からデータを取得し、カテゴリ（Programming / Visual）ごとに分類して表示。
    *   `WorkCard.astro`: 作品のサムネイルカード。
    *   `WorkModal.astro`: `<dialog>` タグを用いた詳細表示モーダル。Markdown 本文（`<Content />`）や動画、関連リンクをレンダリング。
    *   `SkillsList.astro`: 習熟度（★）付きのスキルカードをカテゴリ別に表示。
    *   `AboutSection.astro`: 自己紹介セクション。
    *   `Hero.astro`: 各セクションのタイトルとタグラインを表示。
    *   `ThemeToggle.astro`: ライト/ダークモード切り替えスイッチ。
*   **`src/content/work/`**:
    *   `programming/`, `visual/`: 作品ごとの Markdown ファイル。画像、動画パス、技術スタック、役割などのメタデータを保持。
*   **`src/styles/`**:
    *   `global.css`: サイト全体の配色、共通変数、ダークモード定義。

## 3. 主要な機能と実装のポイント

### A. データ駆動型の Works セクション
制作実績はすべて Markdown で管理されています。
*   **自動フィルタリング:** `classification` フィールドに基づいて「プログラミング」と「ビジュアル」のセクションに自動的に振り分けられます。
*   **モーダル表示:** 作品カードをクリックすると、ページ遷移なしで詳細情報（動画デモ、制作詳細、リンク等）がモーダルで開きます。

### B. スキルセットの可視化
*   `SkillsList.astro` 内で定義されたスキル情報を、カテゴリ（グラフィックス、汎用言語、Web、その他）ごとにフィルタリングして表示。
*   習熟度を星（★）の数で表現し、ホバーアニメーション付きのカード形式でレイアウト。

### C. 動的な背景演出
*   `index.astro` にて、JavaScript を使用してランダムな円（`.bg-circle`）を生成。
*   スクロール量に応じたパララックス（視差効果）アニメーションを実装し、ページに奥行きと動的な質感を与えています。

### D. テーマ切り替え
*   ユーザーのシステム設定（`prefers-color-scheme`）を尊重しつつ、手動での切り替えも可能。
*   `<html>` タグのクラス制御により、CSS 変数を一括で切り替えています。

## 4. デザインと演出
*   **背景:** ノイズテクスチャとグラデーション、さらに動的なパーティクルを組み合わせたリッチな背景。
*   **レスポンシブ:** モバイルファーストの設計で、大画面ではグリッドレイアウト、スマホではシングルカラムに最適化。
*   **アクセシビリティ:** モーダル開閉時のフォーカス管理や、キーボード操作（Enter/Space）によるモーダル起動に対応。
